# cs7641-ml

Requires
========
Conda


Create Conda Environment
========================
1. Clone repository: git clone https://github.com/pdjely/cs7641-ml.git
2. Install conda dependencies: conda env create -f environment.yaml


Activate Environment and Install Additional Deps
================================================
cd cs7641-ml
conda activate cs7641
# Set up additional dependencies
sh ./a4_setup.sh


Run Assignment 4
================
cd src
python assignment4.py
